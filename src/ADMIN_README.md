## 🎨 الصفحات المتوفرة

### ✅ 1. لوحة التحكم الرئيسية (`/admin/dashboard`)
- **الإحصائيات الفورية**:
  - إجمالي المبيعات
  - عدد الطلبات
  - المواعيد الطبية
  - عدد المستخدمين
- **أحدث الطلبات**: جدول بآخر الطلبات مع الحالات
- **المواعيد القادمة**: قائمة بالمواعيد الطبية المؤكدة
- **إحصائيات سريعة**: عدد المنتجات، الزيارات، معدل التحويل

---

### ✅ 2. إدارة المنتجات (`/admin/products`)
**الميزات:**
- ✅ **عرض جميع المنتجات** في جدول تفاعلي
- ✅ **بحث متقدم** بالاسم أو الماركة أو الكود
- ✅ **فلترة حسب التصنيف**: نظارات شمسية، طبية، عدسات لاصقة
- ✅ **إضافة منتج جديد** مع نموذج مفصل
- ✅ **تعديل المنتجات** الموجودة
- ✅ **حذف المنتجات** مع تأكيد
- ✅ **عرض المخزون** مع تنبيهات للمنتجات القليلة
- ✅ **صور المنتجات** مع معاينة

**البيانات المتاحة:**
- الاسم (عربي/إنجليزي)
- التصنيف
- الماركة
- السعر
- الكمية في المخزون
- الحالة (نشط/معطل)
- الصورة

---

### ✅ 3. إدارة الطلبات (`/admin/orders`)
**الميزات:**
- ✅ **عرض جميع الطلبات** مع التفاصيل
- ✅ **بحث بالطلب** أو العميل
- ✅ **فلترة حسب الحالة**: قيد الانتظار، قيد المعالجة، مكتمل، ملغي
- ✅ **تغيير حالة الطلب** مباشرة من الجدول
- ✅ **عرض تفاصيل العميل**: الاسم، البريد الإلكتروني
- ✅ **حذف الطلبات** مع تأكيد
- ✅ **إحصائيات الطلبات**: إجمالي، قيد الانتظار، قيد المعالجة، مكتمل
- ✅ **تصدير البيانات** (Export)

**البيانات المتاحة:**
- رقم الطلب
- بيانات العميل
- عدد المنتجات
- المبلغ الإجمالي
- طريقة الدفع
- الحالة
- التاريخ والوقت

**حالات الطلب:**
1. **Pending** (قيد الانتظار): طلب جديد بانتظار المعالجة
2. **Processing** (قيد المعالجة): جاري تجهيز الطلب
3. **Completed** (مكتمل): تم تسليم الطلب بنجاح
4. **Cancelled** (ملغي): تم إلغاء الطلب

### ✅ 4. إدارة طرق الدفع (`/admin/payment-methods`)
**الميزات:**
- ✅ **عرض جميع طرق الدفع** في بطاقات تفاعلية
- ✅ **تفعيل/تعطيل طرق الدفع** بضغطة زر
- ✅ **إضافة طريقة دفع جديدة**
- ✅ **تعديل طرق الدفع** الموجودة
- ✅ **حذف طرق الدفع** مع تأكيد
- ✅ **عرض رسوم المعالجة** لكل طريقة
- ✅ **إحصائيات**: عدد الطرق النشطة والمعطلة

**طرق الدفع المتاحة:**
1. **بطاقة ائتمان** (Credit Card)
   - رسوم: 2.5%
   - حالة: مفعل
2. **فودافون كاش** (Vodafone Cash)
   - رسوم: مجاناً
   - حالة: مفعل
3. **الدفع عند الاستلام** (Cash on Delivery)
   - رسوم: مجاناً
   - حالة: مفعل
4. **إنستا باي** (InstaPay)
   - رسوم: 1.5%
   - حالة: معطل

**البيانات المتاحة:**
- الاسم (عربي/إنجليزي)
- نوع الطريقة
- الأيقونة
- رسوم المعالجة
- الحالة (مفعل/معطل)
- رقم الهاتف (للمحافظ الإلكترونية)

## 🚀 كيفية الاستخدام

### 1. تسجيل الدخول:
```
1. انتقل إلى: /admin/login
2. أدخل البريد: admin@lotfyoptical.com
3. أدخل كلمة المرور: admin123
4. اضغط "تسجيل الدخول"
```

### 2. التنقل:
- استخدم الـ **Sidebar** للانتقال بين الصفحات
- اضغط على **عرض الموقع** للعودة للموقع الأساسي
- اضغط على **تسجيل الخروج** للخروج من لوحة التحكم

### 3. إدارة المنتجات:
```
1. انتقل إلى: المنتجات
2. اضغط "إضافة منتج"
3. املأ البيانات
4. اضغط "إضافة"
```

### 4. إدارة الطلبات:
```
1. انتقل إلى: الطلبات
2. غيّر حالة الطلب من القائمة المنسدلة
3. احذف طلب بالضغط على أيقونة الحذف
```

### 5. إدارة طرق الدفع:
```
1. انتقل إلى: طرق الدفع
2. فعّل/عطّل طريقة بالضغط على الزر
3. عدّل أو احذف طريقة حسب الحاجة
```
